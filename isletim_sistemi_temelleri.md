 # İŞLETİM SİSTEMİ TEMELLERİ 

## İşletim Sistemi (OS)
  
 Bilgisayar donanımını yöneten ve kullanıcıya çalışma ortamı sağlayan temel yazılımdır.

## Kernel (Çekirdek)

 İşletim sisteminin kalbidir. Donanım ile programlar arasındaki iletişimi sağlar.

## Process (Süreç)

 Çalışmakta olan bir programdır. İşlemcide zaman paylaşımlı olarak çalışır.

## Thread (İş Parçacığı)

 Bir sürecin içinde çalışan daha küçük görev birimidir. Aynı süreçte birden fazla thread olabilir.
 
## Process-Thread Farkı
 Process kendi belleği olan bağımsız bir programdır, thread ise aynı süreç içinde çalışan ve belleği paylaşan daha küçük görev birimidir.

## Multitasking

 Birden fazla programın aynı anda çalışıyormuş gibi görünmesidir.

## Memory Management (Bellek Yönetimi)

 RAM’in hangi program tarafından nasıl kullanılacağını işletim sistemi belirler.

## Virtual Memory (Sanal Bellek)

 RAM yetmediğinde diskin bir kısmının bellek gibi kullanılmasıdır.

## File System (Dosya Sistemi)

 Dosyaların nasıl saklanacağını ve düzenleneceğini belirleyen yapıdır. (NTFS, FAT32, ext4 vb.)

## Device Driver (Sürücü)

 Donanım ile işletim sistemi arasındaki bağlantıyı sağlayan küçük yazılımlardır.

## Interrupt (Kesme)

 Donanımın işlemciye “işim var” diye sinyal göndermesidir.

## Booting (Önyükleme)

 Bilgisayarın açılırken işletim sistemini yükleme sürecidir.

## Scheduler (Zamanlayıcı)

 Hangi sürecin işlemciyi ne zaman kullanacağını belirler.

## Deadlock (Kilitlenme)

 İki sürecin birbirini bekleyip hiçbirinin ilerleyememesi durumudur.

## User Mode / Kernel Mode

 Programların çalıştığı güvenli mod (user) ve çekirdeğin çalıştığı tam yetkili mod (kernel).

## Operating System Types (İşletim Sistemi Türleri)

 Tek kullanıcılı , çok kullanıcılı, gerçek zamanlı, dağıtık, mobil.

## Security (Güvenlik)

 Yetkilendirme, kullanıcı hesapları, şifreleme gibi koruma işlemlerini içerir.
